Intelligence Dashboard - Flow Diagram

Data flow from GitHub to UI

Data Flow Steps

  • 1. Worker loads config from .env
  • 2. Worker fetches GitHub data via API
  • 3. Worker processes and filters data
  • 4. UI requests data from Worker
  • 5. UI renders cards, tabs, panels
  • 6. User interacts (clicks, selects)
  • 7. Auto-refresh every 30 seconds

Key Components

  • GitHub Fetcher: REST API calls
  • Data Processor: Filter + Format
  • Data Handlers: API routes
  • Header: Project dropdown
  • Tabs: Navigation
  • GitHub Panels: Issues + PRs

Multi-Project Support

  • Bug Reporter Dashboard
  • Intelligence Dashboard
  • Paperclip (core)
  • Per-project filtering
  • Repo-specific issue/PR display
